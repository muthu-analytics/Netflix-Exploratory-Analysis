# Netflix Data Analysis
This project analyzes Netflix movies and TV shows using Python and Plotly. 
It visualizes trends by country, genre, release year, and more to provide insights into Netflix content.
## Dataset
- File used: `netflix_titles_nov_2019.csv`
- Columns: show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description
## Features
1. Content type distribution (Movies vs TV Shows)
2. Yearly trends of content added
3. Release year analysis
4. Top categories and genres
5. Country-wise content distribution
6. Top actors and directors
7. Stand-Up Comedy analysis
## Libraries Used
- pandas
- plotly
- collections.Counter
- scipy
